## install

```
bundle install
```

and 

```
npm install
```

## dev

```
bundle exec jekyll serve
```

or

```
bundle exec jekyll serve --drafts
```

and

```
grunt dev
```

## build

```
JEKYLL_ENV=production bundle exec jekyll build
```

and

```
grunt
```

## bump versions

- edit `package.json` and `_config.yml`
